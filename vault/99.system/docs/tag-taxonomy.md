> Daily Note でタグを付ける際は、このリストから選択すること。
> AI（GitHub Copilot）も、このリストに基づいてタグを推論する。

---

## 📌 タグ付けルール

1. **1 項目につき 1〜3 個のタグ**を目安にする
2. **最も具体的なタグ**を優先する（例: `#playwright` > `#testing`）
3. **AI 関連は `#ai` に統一**（mcp, agent, copilot などは `#ai` にまとめる）
4. **新しいタグが必要な場合**は、このファイルに追加してから使用する
