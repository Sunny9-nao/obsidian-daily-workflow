## 1. Obsidian のインストール

1. [Obsidian 公式サイト](https://obsidian.md/) からダウンロード
2. macOS の場合、`.dmg` ファイルを開いてアプリケーションフォルダにドラッグ
3. Obsidian を起動

---

## 2. Vault を開く

1. Obsidian を起動
2. 「Open folder as vault」を選択
3. このリポジトリの `vault/` フォルダを選択
4. 「Open」をクリック

---

## 3. Dataview プラグインのインストール

> **重要**: Dataview はこのワークフローの中核です。必ずインストールしてください。

### 手順

1. **設定画面を開く**
   - macOS: `Cmd + ,`
   - Windows/Linux: `Ctrl + ,`

2. **コミュニティプラグインを有効化**
   - 左メニュー「Community plugins」をクリック
   - 「Turn on community plugins」をクリック
   - 確認ダイアログで「Turn on community plugins」を選択

3. **Dataview をインストール**
   - 「Browse」をクリック
   - 検索ボックスに「Dataview」と入力
   - 「Dataview」を選択して「Install」をクリック
   - インストール後「Enable」をクリック

4. **Dataview の設定（任意）**
   - 左メニュー「Dataview」をクリック
   - 以下を確認/設定：

   | 項目 | 推奨設定 |
   |------|----------|
   | Enable JavaScript Queries | ON（高度なクエリ用） |
   | Enable Inline Queries | ON |
   | Refresh Interval | 2500ms（デフォルト） |

## 4. デイリーノートの設定

### 手順

1. **設定画面を開く**
   - macOS: `Cmd + ,`

2. **Core plugins → Daily notes を有効化**
   - 左メニュー「Core plugins」をクリック
   - 「Daily notes」のトグルを ON

3. **Daily notes の設定**
   - 左メニュー「Daily notes」をクリック
   - 以下のように設定：

   | 項目 | 設定値 |
   |------|--------|
   | New file location | `00.daily` |
   | Template file location | `00.daily/Templates/Daily Note Simple.md` |
   | Date format | `YYYY-MM-DD` |
   | Open daily note on startup | お好みで |

---

## 5. テンプレートの設定

### 手順

1. **Core plugins → Templates を有効化**
   - 左メニュー「Core plugins」をクリック
   - 「Templates」のトグルを ON

2. **Templates の設定**
   - 左メニュー「Templates」をクリック
   - 以下のように設定：

   | 項目 | 設定値 |
   |------|--------|
   | Template folder location | `00.daily/Templates` |
   | Date format | `YYYY-MM-DD` |
   | Time format | `HH:mm` |

### 使い方

- ノート内で `Cmd + P`
- 「Templates: Insert template」を選択
- 使いたいテンプレートを選択

## 6. 索引ページの使い方

### 📁 索引ファイルの場所

```
vault/
  01.index/
    00.dashboard.md    ← 全体概要
    01.learnings.md    ← 学び一覧
    02.ideas.md        ← アイディア一覧
    03.actions.md      ← 未完了アクション
    04.experiments.md  ← 実験一覧
```

### 使い方

1. **索引ページを開く**
   - `01.index/` フォルダ内の任意のファイルを開く
   - Dataview が動的に一覧を表示

2. **項目をクリック**
   - 各項目をクリックすると元の Daily Note にジャンプ

3. **アクションを完了にする**
   - `03.actions.md` で未完了アクションを確認
   - 元の Daily Note を開いて `- [ ]` を `- [x]` に変更
   - 保存すると actions.md から自動的に消える

## 7. 推奨設定

### エディター設定

| 項目 | 推奨設定 |
|------|----------|
| Default view for new tabs | Reading view（閲覧時は見やすく） |
| Strict line breaks | ON（改行を維持） |
| Readable line length | ON（読みやすい行幅） |

### ファイル設定

| 項目 | 推奨設定 |
|------|----------|
| Default location for new notes | `00.daily` |
| Default location for new attachments | `99.system/attachments` |

### タグ設定

- 左サイドバーの「Tags」パネルを活用
- タグをクリックすると該当ノートを一覧表示
